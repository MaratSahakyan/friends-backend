    # Installation

1. Create copy of `.env.example` file as `.env`
2. Install docker desktop, if not installed.
3. Run postgres with Docker.
    ```bash
   npm run docker
   ```
4. Run this command.
    ```bash
    npm install
    ```
5. Run this command.
    ```bash
    npm run build
    ```
6. Running the app locally
    ```bash
    npm run start:dev
    ```
